# Service Control Policy catalog

`control-catalog.json` is the machine-readable native-control and custom-gap
mapping. It intentionally records published control names and stable documented
identifiers, not guessed Control Catalog opaque ARNs.

At deployment time, resolve the current global ARN with the AWS Control Catalog
`ListControls`/`GetControl` APIs and confirm the implementation and behavior
still match the reviewed entry. Do not enable a native control or attach a custom
SCP solely from this static file.

Custom policy JSON is generated by `infra/modules/scp-guardrails`. This avoids
maintaining a second copy that could drift from Terraform. Use the module outputs
or Terraform plan JSON with `scripts/validate-scp.sh` for validation.
